Hard Drive

Hard drives take many forms, such as a spinning disk (HDD), or a solid state drive, which requires no moving parts, and is faster in most cases.

Cache

Cache is high speed memory, used to store frequently used data so it can be quickly retrieved by the user or application.

RAM

RAM is memory that temporarily stores data and instructions. It can also be reffered to as primary or main memory. It is often very high speed, and it will lose all data without power.

ROM

ROM stands for 'Read Only Memory', and stores data that is permanant and is part of the computer, including the BIOS.

Motherboard

The motherboard is the main circuit board inside of the computer. The motherboard contains circuits called 'buses', for transporting information. It connects all components in the computer.

CPU

The CPU is the 'Central Processing Unit' of the computer. It processes data, and controls the flow of data between the other units. It also contains the ALU (Arithmetic Logic Unit)
